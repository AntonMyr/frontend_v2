## How to install
### Requirements
| Technologies  | Version       
| ------------- |:-------------:|
| Nodejs        | v8.9.4+       |
| Npm           | 5.6.0         |

### Setup the frontend project
1. `cd /frontend && npm install && npm start`
Your project should start a new chrome window.
